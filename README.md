# 點點英雄 · 禮品碼批量兌換工具

一個讓你一次兌換多組禮品碼的網頁工具，省去反覆輸入玩家 ID 的麻煩。

🔗 **線上使用：** https://qpalzm963.github.io/dicero-batch-redeem/

---

## 使用方式

1. 開啟工具頁面
2. 輸入你的**玩家 ID**（支援 `12345` 或 `1-12345678` 兩種格式）
3. 在文字框貼上多組兌換碼，**每行一組**
4. 點「開始兌換」，依序輸入每組的驗證碼即可

## 功能

- 玩家 ID 只需輸入一次
- 兌換碼自動排隊逐一處理
- 驗證碼錯誤時自動刷新
- 每組可單獨跳過
- 顯示每組兌換結果（成功 / 失敗原因 / 已跳過）

## 注意事項

- 每組兌換碼仍需手動輸入一次驗證碼（遊戲官方限制，無法自動化）
- 若出現 CORS 錯誤，請直接在官方兌換頁面的瀏覽器 Console 中執行
